## Prompt Replit Agent

### Tujuan
Buat aplikasi web mobile-first bernama **Self Tracker** dengan integrasi penuh ke Google Sheets (Apps Script Web App), mendukung fallback ke LocalStorage saat offline.

### Stack
- HTML + CSS (utility classes ala Tailwind, boleh pakai CDN)
- Vanilla JS (modular, file terpisah per fungsi)
- Chart.js untuk visualisasi grafik
- Storage: Google Sheets (utama), LocalStorage (fallback)

### Struktur Project
- index.html
- /assets/css/style.css
- /assets/js/app.js           // UI logic
- /assets/js/storage.js       // LocalStorage + merge
- /assets/js/api.js           // fetch wrapper, sync logic
- /assets/js/charts.js        // setup & render Chart.js
- /assets/js/renderers.js     // builder grid & tabel

### Google Sheets Integration
- Gunakan Google Apps Script Web App
- Token auth via header: Authorization: Bearer ${API_TOKEN}
- CORS aktif
- Endpoint:
  - GET /health
  - POST /init
  - POST /sync/pull
  - POST /sync/push

### Apps Script Backend
- Autogenerate script untuk:
  - Buat sheet & header jika belum ada
  - Auth check
  - Routing berdasarkan endpoint
  - Upsert & pull data dengan updated_at
  - Response selalu JSON
- Instruksi deploy:
  - Buka script.google.com â†’ New Project
  - Paste kode, set TOKEN & SPREADSHEET_ID
  - Deploy as Web App (execute as: Me, who has access: Anyone with link)
  - Salin Deployment ID ke API_BASE di frontend
